Privacy Policy

Last Updated: January 1, 2025 | Effective Date: January 1, 2025 한국어 버전

Where We Are (hereinafter referred to as "the App") values your privacy and complies with applicable data protection laws, including the Personal Information Protection Act.

1. Purpose of Collection and Use of Personal Information

The App collects the following personal information to provide real-time location sharing services between friends:

1.1 Types of Personal Information Collected

  • Location Information: GPS-based real-time location (latitude, longitude, accuracy, speed)
  • User Identification Information: Device unique identifier (Device ID), push notification token
  • User Settings Information: Nickname, group code (Topic)
  • Device Information: Transmission time

1.2 Methods of Collection

  • Direct input by users during app usage
  • Automatic collection when location permission is granted
  • Message reception through push notification services

2. Purpose of Use of Personal Information

Collected personal information is used only for the following purposes:

  • Providing Location Sharing Services: Real-time location sharing with friends using the same group code (Topic)
  • Processing Location Requests/Responses: Automatic response to friends' location requests
  • Service Quality Improvement: App error analysis and service improvement

3. Retention and Use Period of Personal Information

3.1 Real-time Location Information

  • Server Transmission: No separate storage on server; used only for transmission
  • Local Device Storage: Stored to display friends' last known location. Automatically deleted after maximum 24 hours of cache storage
  • No Long-term Storage: Location information is not stored in any database

3.2 User Settings Information

  • Device Storage: Retained until app deletion
  • No Server Storage: Nicknames and location information are temporarily stored for display purposes only and not saved in any separate database

4. Provision of Personal Information to Third Parties

The App does not provide personal information to third parties, except in the following cases:

  • Users with Same Group Code: Location information is shared in real-time with friends using the same group code (with explicit user consent)
  • Legal Requirements: When required by law or necessary to comply with legal obligations

Important: Sharing a group code (Topic) means sharing your location information with all users in that group. Please share group codes only with people you trust.

5. Destruction Procedures and Methods of Personal Information

5.1 Automatic Destruction

  • Server Location Data: No separate storage on server (used only for message transmission)
  • Local Cache: Automatically deleted after 24 hours

5.2 Manual Destruction

  • App Deletion: All information stored on the device is immediately deleted when the app is uninstalled
  • Settings Reset: All friend location information can be deleted using the 'Clear All Friends' function in app settings

6. Security Measures for Personal Information

The App implements the following measures to ensure the security of personal information:

  • Encrypted Communication: Safe data transmission using HTTPS/TLS protocol
  • Minimum Permissions: Collection of only the minimum information necessary for service provision
  • Automatic Deletion: Automatic deletion system after retention period (24 hours)
  • Access Control: Location information sharing only possible for users who know the group code

7. User Rights and How to Exercise Them

Users have the following rights and can exercise them at any time:

  • Stop Location Sharing: Disable 'Auto-send Location' in app settings
  • Revoke Location Permission: Remove permission in iOS Settings > Privacy & Security > Location Services
  • Change Group Code: Automatically leave existing group when changing group code in app settings
  • Delete App: All information on device is immediately deleted upon app deletion

8. Location Information Terms of Use

8.1 Consent for Location Information Collection

The App collects users' location information to provide location-based services. Location information collection occurs only when users explicitly consent during app installation.

8.2 Purpose of Use or Provision of Location Information

  • Real-time location sharing between friends
  • Location display on maps
  • Location-based notification services

8.3 Recipients of Location Information

Provided only to users using the same group code (Topic).

8.4 Timing of Location Information Provision

  • When user presses the 'Check Location' button
  • When receiving a friend's location request with 'Auto-send Location' setting enabled

9. Protection of Children's Personal Information

The App does not collect personal information from children under 14 years of age. For children under 14 to use the app, consent from a legal guardian is required.

10. Changes to Privacy Policy

This Privacy Policy may be modified according to changes in relevant laws or services. Changes will be announced in advance through in-app notices, and important changes will be announced at least 7 days in advance.

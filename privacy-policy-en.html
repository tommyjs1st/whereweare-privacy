Privacy Policy

Last Updated: January 1, 2025 | Effective Date: January 1, 2025 한글 버전

Where We Are (the "App") values your privacy and complies with applicable privacy laws and regulations.

1. Purpose of Collecting and Using Personal Information

The App collects the following personal information to provide real-time location sharing services between friends:

1.1 Types of Personal Information Collected

  • Location Information: GPS-based real-time location (latitude, longitude, accuracy, speed)
  • User Identification: Device unique identifier (Device ID), push notification token
  • User Settings: Nickname, Call sign (Topic)
  • Device Information: Transmission timestamp

1.2 Methods of Collection

  • Directly entered by users during app usage
  • Automatically collected when location permission is granted
  • Received through push notification service

2. Purpose of Using Personal Information

The collected personal information is used solely for the following purposes:

  • Location Sharing Service: Real-time location sharing with friends using the same call sign (Topic)
  • Location Request/Response Processing: Automatic response to friends' location requests
  • Service Quality Improvement: App error analysis and service enhancement

3. Retention and Use Period of Personal Information

3.1 Real-time Location Information

  • Server Transmission: Not stored on the server; only used for message relay
  • Local Device Storage: Cached for up to 24 hours to display friends' last known locations, then automatically deleted
  • No Long-term Storage: Location information is not stored in any database

3.2 User Settings Information

  • Device Storage: Retained until the app is deleted
  • No Server Storage: Nickname and call sign are stored only on the device and not transmitted to servers

4. Provision of Personal Information to Third Parties

The App does not provide personal information to third parties, except in the following cases:

  • Users with Same Call Sign: Location information is shared in real-time with friends using the same call sign (with user's explicit consent)
  • Legal Requirements: When required by law or necessary to comply with legal obligations

Important: Sharing a call sign (Topic) means sharing your location information with all users in that group. Only share your call sign with people you trust.

5. Destruction Procedure and Method

5.1 Automatic Destruction

  • Server Location Data: No separate storage on server (used only for message relay)
  • Local Cache: Automatically deleted after 24 hours

5.2 Manual Destruction

  • App Deletion: All information stored on the device is immediately deleted when the app is uninstalled
  • Settings Reset: All friend location information can be deleted using the 'Delete All Friends' feature in the app

6. Security Measures

The App takes the following measures to ensure the security of personal information:

  • Encrypted Communication: Secure data transmission using HTTPS/TLS protocol
  • Minimal Permissions: Collecting only the minimum information necessary for service provision
  • Automatic Deletion: Operating an automatic deletion system when the retention period expires
  • Access Control: Location information accessible only to users who know the call sign

7. User Rights and How to Exercise Them

Users have the following rights and can exercise them at any time:

  • Stop Location Sharing: Disable 'Auto-send My Location' in app settings
  • Revoke Location Permission: Remove permission in iOS Settings > Privacy > Location Services
  • Change Call Sign: Changing the call sign in app settings automatically exits from the existing group
  • Delete App: All information on the device is immediately deleted when the app is uninstalled

8. Location Information Terms of Use

8.1 Consent to Location Information Collection

The App collects user location information to provide location-based services. Location information collection only occurs when users explicitly consent during app installation.

8.2 Purpose of Using or Providing Location Information

  • Real-time location sharing between friends
  • Displaying locations on the map
  • Location-based notification services

8.3 Recipients of Location Information

Provided only to users using the same call sign (Topic).

8.4 Timing of Location Information Provision

  • When the user presses the 'Check Location' button
  • When receiving a friend's location request with 'Auto-send My Location' enabled

9. Children's Privacy Protection

The App does not collect personal information from children under 14 years of age. Consent from a legal guardian is required for children under 14 to use the app.

10. Changes to Privacy Policy

This privacy policy may be revised in accordance with changes in relevant laws or services. Changes will be announced in advance through in-app notifications, and important changes will be announced at least 7 days in advance.

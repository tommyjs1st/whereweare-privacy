개인정보 처리방침

최종 수정일: 2025년 1월 1일 | 시행일: 2025년 1월 1일

Where We Are(이하 "본 앱")는 사용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 등 관련 법령을 준수하고 있습니다.

1. 개인정보의 수집 및 이용 목적

본 앱은 친구 간 실시간 위치 공유 서비스를 제공하기 위해 다음과 같은 개인정보를 수집합니다:

1.1 수집하는 개인정보 항목

  • 위치 정보: GPS 기반 실시간 위치 (위도, 경도, 정확도, 속도)
  • 사용자 식별 정보: 기기 고유 식별자(Device ID), 토큰
  • 사용자 설정 정보: 별명(닉네임), 호출부호(Topic)
  • 기기 정보: 전송 시간

1.2 개인정보의 수집 방법

  • 앱 사용 과정에서 사용자가 직접 입력
  • 위치 권한 허용 시 자동 수집
  • 푸시 알림 서비스를 통한 메시지 수신

2. 개인정보의 이용 목적

수집된 개인정보는 다음의 목적으로만 이용됩니다:

  • 위치 공유 서비스 제공: 동일한 호출부호(Topic)를 사용하는 친구들과 실시간 위치 공유
  • 위치 요청/응답 처리: 친구의 위치 요청에 대한 자동 응답
  • 서비스 품질 개선: 앱 오류 분석 및 서비스 개선

3. 개인정보의 보유 및 이용 기간

3.1 실시간 위치 정보

  • 서버를 통한 전송: 서버내 별도 저장은 하지 않으며, 위치 전송시 서버를 통해 전송
  • 기기 로컬 저장: 친구를 최종위치를 표시하기 위해 저장. 최대 24시간 캐시 저장 후 자동 삭제
  • 장기 보관 없음: 위치 정보는 데이터베이스에 저장되지 않습니다

3.2 사용자 설정 정보

  • 기기 내 저장: 앱을 삭제할 때까지 보관
  • 서버 저장 없음: 별명, 호출부호 등은 기기에만 저장되며 서버로 전송되지 않습니다

4. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 단, 다음의 경우는 예외로 합니다:

  • 동일 호출부호 사용자: 같은 호출부호를 사용하는 친구들에게 위치 정보가 실시간으로 공유됩니다 (사용자의 명시적 동의)
  • 법령에 의한 요구: 법령에 특별한 규정이 있거나 법령상 의무를 준수하기 위해 불가피한 경우

중요: 호출부호(Topic)를 공유한다는 것은 해당 그룹의 모든 사용자와 위치 정보를 공유한다는 의미입니다. 신뢰할 수 있는 사람들과만 호출부호를 공유하시기 바랍니다.

5. 개인정보의 파기 절차 및 방법

5.1 자동 파기

  • 서버 위치 데이터서버내 별도 저장 없음(메세지 전달시에만 사용)
  • 로컬 캐시: 24시간 경과 시 자동 삭제

5.2 수동 파기

  • 앱 삭제: 앱을 삭제하면 기기에 저장된 모든 정보가 즉시 삭제됩니다
  • 설정 초기화: 앱 내 '설정 초기화' 기능으로 모든 친구위치 정보 삭제 가능

6. 개인정보의 안전성 확보 조치

본 앱은 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다:

  • 암호화 통신: HTTPS/TLS 프로토콜을 사용한 안전한 데이터 전송
  • 최소 권한: 서비스 제공에 필요한 최소한의 정보만 수집
  • 자동 삭제: 보유 기간 경과 시 자동 삭제 시스템 운영
  • 접근 제한: 호출부호를 알고 있는 사용자만 위치 정보 접근 가능

7. 사용자의 권리와 행사 방법

사용자는 다음과 같은 권리를 가지며, 언제든지 행사할 수 있습니다:

  • 위치 공유 중단: 앱 설정에서 '자동 응답' 비활성화
  • 위치 권한 철회: iOS 설정 > 개인정보 보호 > 위치 서비스에서 권한 제거
  • 호출부호 변경: 앱 설정에서 호출부호 변경 시 기존 그룹에서 자동 탈퇴
  • 앱 삭제: 앱 삭제 시 기기 내 모든 정보 즉시 삭제

8. 위치 정보 이용 약관

8.1 위치 정보 수집 동의

본 앱은 위치기반서비스를 제공하기 위해 사용자의 위치 정보를 수집합니다. 위치 정보 수집은 사용자가 앱 설치 시 명시적으로 동의한 경우에만 이루어집니다.

8.2 위치 정보의 이용 또는 제공 목적

  • 친구 간 실시간 위치 공유
  • 지도 상 위치 표시
  • 위치 기반 알림 서비스

8.3 위치 정보 제공 대상

동일한 호출부호(Topic)를 사용하는 사용자에게만 제공됩니다.

8.4 위치 정보 제공 시점

  • 사용자가 '위치 확인하기' 버튼을 누른 경우
  • '자동 응답' 설정이 활성화된 상태에서 친구의 위치 요청을 받은 경우

9. 어린이의 개인정보 보호

본 앱은 만 14세 미만 어린이의 개인정보를 수집하지 않습니다. 만 14세 미만 어린이가 앱을 사용하려면 법정대리인의 동의가 필요합니다.

10. 개인정보 처리방침의 변경

본 개인정보 처리방침은 관련 법령 또는 서비스의 변경에 따라 수정될 수 있습니다. 변경 시 앱 내 공지사항을 통해 사전 공지하며, 중요한 변경사항이 있을 경우 최소 7일 전에 공지합니다.
